### 2. Full Features (What the app go do)

**MVP Features (Must build — Core of the project)**  
1. User registration & login (customers + restaurant admins)

3. Menu management (add categories + food items with price & description)  
4. Cart system (add/remove items)  
5. Place order + calculate total  
6. Paystack payment integration (initialize payment + verify via webhook)

- JWT authentication with refresh tokens  
- Basic analytics endpoint (e.g. total orders per restaurant)  
- Idempotency for payments (no double charging)  
- Proper error handling & logging  
- Docker + docker-compose for easy run  
- Unit + integration tests

